#  الحاسبة الاحترافية للعمر (Professional Age Calculator)

مشروع تطبيقي لصفحة ويب تفاعلية بلغة PHP، تم تطويره كجزء من رحلة تعلم لتطبيق مفاهيم البرمجة وفصل الاهتمامات (MVC).

هذا المشروع يحسب العمر بدقة بناءً على تاريخ الميلاد، ويقدم ميزات إضافية مثل العد التنازلي لعيد الميلاد القادم.

##  الميزات الحالية

* **حساب دقيق للعمر:** يعرض العمر بالسنوات والشهور والأيام.
* **العد التنازلي:** يحسب عدد الأيام المتبقية حتى عيد الميلاد القادم.
* **تصميم احترافي:** واجهة مستخدم نظيفة (UI) وتجربة مستخدم (UX) سلسة باستخدام HTML5 و CSS3.
* **بنية قوية (Robust Backend):**
    * استخدام **PHP** لمعالجة المنطق في الخادم.
    * **فصل الاهتمامات:** فصل منطق PHP (`index.php`) عن العرض (`view.php`) والتصميم (`style.css`).
    * **معالجة الأخطاء:** التحقق من التواريخ المدخلة (مثل عدم إدخال تاريخ مستقبلي).

##  التثبيت والتشغيل

هذا المشروع يعتمد على PHP، لذلك **لا يمكن تشغيله بالنقر المزدوج** على الملفات. يجب تشغيله عبر خادم ويب محلي.

**المتطلبات:**
* خادم ويب يدعم PHP (مثل **XAMPP** أو MAMP أو WAMP).

**خطوات التشغيل:**

1.  **تحميل الخادم:** قم بتحميل وتثبيت برنامج [XAMPP](https://www.apachefriends.org/index.html).
2.  **تحميل المشروع:** قم بتحميل هذا المشروع (أو استخدم `git clone`).
3.  **نقل الملفات:** انسخ مجلد المشروع بالكامل إلى مجلد `htdocs` داخل مجلد تثبيت XAMPP.
    * *مثال المسار على ويندوز:* `C:\xampp\htdocs\age-calculator`
4.  **تشغيل الخادم:** افتح لوحة تحكم XAMPP واضغط **Start** بجانب **Apache**.
5.  **فتح المشروع:** افتح المتصفح واكتب الرابط التالي (استبدل `age-calculator` باسم مجلدك):
    ```
    http://localhost/age-calculator/
    ```

##  خارطة الطريق (Roadmap)

الميزات المخطط لتطويرها مستقبلاً:

* **المرحلة 1 (تفاعلية):** استخدام JavaScript (AJAX) لجعل الحساب فوري **بدون إعادة تحميل** الصفحة.
* **المرحلة 2 (التوسع):**
    * إضافة أدوات جديدة (حاسبة الفارق بين تاريخين).
    * إعادة الهيكلة باستخدام **OOP** لجعل الكود أكثر احترافية.
* **المرحلة 3 (الاحترافية):**
    * إضافة قاعدة بيانات (MySQL) لحفظ تواريخ الميلاد.
    * بناء **API** منفصلة للواجهة الخلفية.
